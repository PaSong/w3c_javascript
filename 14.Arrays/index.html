Arrays

자바스크립트 배열은 단일 변수에 여러 값을 저장하는 데 사용

var cars = ["Saab", "Volvo","BMW"];

배열이란?

배열은 한 번에 둘 이상의 값을 저장 할 수 있는 특수 변수

배열 만들기

배열 리터럴을 사용하는 것이 자바스크립트 배열을 만드는 가장 쉬운 방법

var array_name = [item1,item2,...];

공백과 줄바꿈은 중요하지 않음

자바스크립트 키워드 사용하기 new

var cars = new Array("Saab","Volvo","BMW");

new Array를 사용할 필요 없음

배열 요소 액세스

인덱스 번호를 참조하여 배열 요소에 엑세스함.

var name = cars[0]

배열 요소 변경

var cars = ["Saab","Volvo","BMW"]; cars[0] = "Opel"; documnet.getElementById("demo").innnerHTML = cars[0];

전체 배열 엑세스

JavaScript를 사용하면 배열 이름을 참조하여 전체 배열에 액세스 할 수 있음.

배열은 객체임.

배열은 특별한 유형의 객체. JavaScript의 typeOf 연산자는 'object'를 반환

배열 요소가 객체가 될 수 있음.

자바스크립트 변수는 객체가 될 수 있음. 배열은 특별한 종류의 객체

이 때문에 동일한 배열에서 여러 유형의 변수를 가질 수 있음.

배열은 객체를 가질 수 있음. 배열에 함수를 포함 할 수 있음. 배열에 배열을 가질 수 있음.

배열 속성 및 메서드

자바스크립트 배열으 진정한 강점은 내장 배열 속성 및 메서드임.

var x = car.length; // The length property returns the number of elements

\ length 속성은 항상 가장 높은 배열 인덱스보다 하나 더 큼

var y = cars.sort(); // The sort() method sorts arrays

마지막 배열 요소 액세스

var last = fruits[fruits.length -1];
