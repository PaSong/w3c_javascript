HTML DOM 요소

이 페이지는 HTML 페이지에서 HTML요소를 찾아 액세스하는 방법을 알려줌

HTML 요소 찾기

JavaScript를 사용하여 HTML요소를 조작하려고 함

이렇게 하려면 요소를 먼저 찾아야함. 이를 수행하는 방법에는 여러 가지가 있음

  • ID로 HTML 요소찾기
  • 태그 이름으로 HTML 요소 찾기
  • 클래스 이름으로 HTML 요소 찾기
  • CSS 선택자로 HTML 요소 찾기
  • HTML 개체 컬렉션을 사용하여 HTML요소 찾기

ID로 HTML요소 찾기

DOM에서 HTML요소를 찾는 가장 쉬운 방법은 요소 id를 사용하는 것

var myElement = document.getElementById("intro"); document.getElementById("demo").innerHTML = "The text from the intro paragraph is" + myElement.innerHTML;

태그 이름으로 HTML요소 찾기

모든 p요소를 찾음

var x = document.getElementByTagName("p"); document.getElementById("demo").innerHTML = 'The text in first paragraph (index 0) is:' + x[0].innerHTML;

클래스 이름으로 HTML 요소 찾기

동일한 클래스 이름을 가진 모든 HTML 요소를 찾으려면 사용함

var x = document.getElementByClassName("intro"); document.getElementById("demo").innerHTML = 'The first paragraph (index 0) with class ="intro":' + x[0].innerHTML;

Internet Explorer8 및 이전 버전에서는 클래스 이름별로 요소를 찾지 못함

css 선택자로 HTML 요소 찾기

지정된 css선택기(ID, 클래스 이름, 유형, 속성, 속성 값 등)와 일치하는 모든 HTML 요소를 찾으려면 querySelectorAll()메소드를 사용해라

var x = document.querySelectorAll("p.intro"); document.getElementById("demo").innerHTML = 'The first parargraph (index 0) with class = "intro":'+ x[0].innerHTML; //intro 클래스가 있는 p태그를 찾음

querySelectorAll()메서드는 Internet Explorer 8 및 이전 버전에서는 작동하지 않음
