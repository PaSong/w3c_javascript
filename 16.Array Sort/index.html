배열 정렬

sort() 메소드는 알파벳 배열을 정렬함

배열 반전

reverse()메소드는 배열의 요소를 반대로 정렬(내림차순으로)

숫자 정렬

기본적으로 sort() 함수는 값은 문자열로 정렬함

그러나 숫자를 문자열로 정렬하면 "2"가 "1"보다 크기 때문에 "25"는"100"보다 크다고 나옴(이것이 문제임)

이 문제를 해결하려면 비교기능을 사용해야함

오름차순 var points =[40,100,1,5,25,10]; points.sort(function(a,b){return a-b})

내림차순 var points = [40,100,1,5,25,10]; points.sort(function(a,b){return b-a});

비교함수

compare 함수의 목적은 대체 정렬 순서를 정의하는 것

compare 함수는 인수에 따라 음수, 0 또는 양수 값을 반환해야함

40과 100을 비교할 때 sort()메서드는 compare 함수 (40,100)를 호출함. 이 함수는 40-100을 계산하고 -60(음수 값)을 반환함. 그러면 sort함수는 40을 100보다 낮은 값으로 정렬.

무작위 순서로 배열 정렬

var points =[40,100,1,5,25,10]; points.sort(function(a,b){return 0.5 - Math.random()});

가장 높은(또는 가장 낮은) 배열 값 찾기

배열에서 최대 또는 최소값을 찾는 기본 제공 함수는 없음

그러나 배열을 정렬 한 후에 인덱스를 사용하여 가장 높은 값과 가장 낮은 값을 얻을 수 있음.

가장 높은(또는 가장 낮은)값을 찾고자 할 때 전체 배열을 정렬하는 것은 매우 비효율적인 방법.

배열에서 Math.max()사용

Math.max.apply를 사용하여 배열에서 가장 높은 숫자를 찾을 수 있음

function myArrayMax(arr){ return Math.max.apply(null, arr); }

배열에서 Math.min()사용

Math.min.apply를 사용하여 배열에서 가장 낮은 수를 찾을 수 있음.

function myArrayMax(arr){ return Math.min.apply(null, arr); }

나의 최소/최대 javaScript 방법

최대 찾기 function myArrayMax(arr){ var len = arr.length; var max = -Infinity; while (len--){ if(arr[len] > max){ max = arr[len]; } } return max; }

최소 찾기 function myArrayMax(arr){ var len = arr.length; var = -Infinity; while (len--){ if(arr[len] > min){ min = arr[len]; } } return min; }
