자바스크립트 구문

자바 스크립트 값

고정값과 가변 값으로 나뉨

고정값:리터럴, 변수 값:변수

자바 스크립트 리터럴

숫자는 십진법의 유무와 과계없이 쓰여짐

문자열은 이둥 따옴표 또는 작은 따옴표로 작성된 텍스트

자바 스크립트 변수

변수는 데이터 값을 저장하는데 사용

var z; z=6;

자바 스크립트 연산자

산술 연산자를 사용하여 값을 계산

할당 연산자(=)를 사용 하여 변수에 값을 할당

자바스크립트 표현식

표현식은 값으로 계산되는 값, 변수 및 연산자의 조합

계산을 평가라고 함

5*10은 50으로 평가

표현식에는 변수 값이 포함될 수도 있습니다.

자바 스크립트 키워드

키워드는 수행 할 작업을 식별하는데 사용

var에 키워드는 변수를 만들 수 있는 브라우저를 알려줌

자바 스크립트 식별자

숫자는 첫 번째 문자로 사용할 수 없음

대소 문자를 구분함

lastName 와 lastname은 두 개의 다른 변수입니다.

자바 스크립트와 낙타의 경우

자바 스크립트에서는 하이픈을 사용할 수 없습니다.뺄셈을 위해 예약되어 있습니다.
