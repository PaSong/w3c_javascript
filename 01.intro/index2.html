javaScript가 html 내용을 변경할 수 있습니다.

html 메소드 getElementById()

innerHTML: 요소 내용을 변경

document.getElementById("demo").innerHTML = "Hello JavaScript" 이 예제는 메소드를 사용하여 HTML요소를 검색하고 요소 내용을 변경한는것 JavaScript는 큰 따옴표와 작은 따옴표를 모두 허용합니다.

javaScript가 HTML 속성 값을 변경할 수 있습니다.

document.getElementById('myImage').src='파일명'

javaScript는 HTML 스타일 css를 변경할 수 있습니다.

document.getElementById('myImage').style.fontSize="35px";

javaScript는 HTML 요소를 숨기거나 표시할수 있다.

document.getElementById("demo").style.display="none";

document.getElementById("demo").style.display="block";
