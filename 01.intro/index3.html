JavaScript

JavaScript는 HTML의 기본 스크립팅 언어입니다.

head 또는 body의 자바스크립트

HTML 문서에 원하는 수의 스크립트를 배치 할 수 있습니다.

외부 JavaScript 장점

html과 코드 구분함.

html과 JavaScript를 읽고 유지하기가 더 쉬움

캐시 된 JavaScript파일로 인해 페이지로드 속도가 빨라질 수 있음.

외부 참조(경로)

/../....js : 현재 웹 사이트의 지정된 폴더에 있는 스크립트를 사용

....js : 현재 페이지와 같은 폴더에 있는 스크립트에 연결
