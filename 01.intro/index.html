자바 스크립트 자습서

JavaScript는 HTML과 웹의 프로그래밍 언어입니다.

왜 javaScript를 배워야하는가?

자바스크립트는 웹 개발자가 반드시 알아야 하는 3가지 언어중 하나입니다.

1.웹 페이지의 내용을 정의하는 HTML

2.웹 페이지의 레이아웃을 지정하는 CSS

3.웹 페이지의 동작을 프로그래밍하는 JavaScript

웹 페이지는 JavaScript가 사용되는 유일한 장소가 아닙니다. 많은 데스크탑 및 서버 프로그램은 JavaScript를 사용합니다. node.js가 가장 잘 알려져 있다. MongoDB및 CouchDB와 같은 일부 테이터베이스는 JavaScript를 프로그래밍언어로 사용.
