객체

모든 자동차는 동일한 속성을 갖지만 속성 값은 자동차마다 다름.

JavaScript 변수는 데이터 값의 컨테이너라는 것을 배움

객체도 변수. 객체에는 많은 값이 포함 될 수 있음

var car ={type:"Fiat",model:"500",color:"pink"};

자바스크립트 객체는 속성 또는 메서드라고 하는 명명 된 값의 컨테이너

객체 정의

객체 리터럴을 사용하여 자바스크립트의 객체를 정의 (및 생성)합니다

공백과 줄 바꿈은 중요하지 않음

객체 속성

이름:값 = 속성

객체 속성 액세스

두 가지 방법으로 객체 속성에 액세스

objectName.propertyname

objectName["propertyName"]

객체 메소드

객체는 메소드를 가질 수 도 있음.

메서드는 객체에서 수행 할 수 있는 작업

메소드는 함수 정의로 특성에 저장

메소드는 속성으로 저장된 함수

객체 메소드 액세스

objectName.methodName()

name=person.fullName()

but ()괄호 없이 메서드에 엑세스 하면 함수 정의가 반환.

문자열 ,숫자 및 부울을 객체로 선언하지 마라!

콛즈가 복잡해지고 실행 속도가 느려짐.
