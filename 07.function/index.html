함수

함수 구문

함수 이름에는 문자,숫자,밑줄 및 달러 기호(번수와 동일한 규칙)가 포함될 수 있음.

괄호는 매개 변수 이름을 쉼표로 구분하여 포함 할 수 있음.

함수 반환

자바스크립트가 return문에 도달 하면 함수의 실행이 중지됨.

함수가 명령문에서 호출 된 경우 JavaScript는 호출 문 다음에 코드를 실행하기 위해 return함.

기능?

코드를 재사용 할 수 있음.

()연산자는 함수를 호출함

toCelsius는 함수 객체를 참조

toCelsius()는 함수 결과를 참조

변수 값으로 사용되는 함수

함수를 변수 값으로 직접 사용할 수 있음.

지역번수

함수 내에서 선언된 변수는 함수의 LOCAL이 됨.

로컬 변수는 함수 내에서만 액세스 할 수 있음.
