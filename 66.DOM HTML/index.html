HTML 변경

HTML DOM을 사용하면 JavaScript가 HTML 요소의 내용을 변경할 수 있음

HTML 출력 스트림 변경하기

자바스크립트는 동적 HTML 콘텐츠를 생성 할 수 있음

JavaScript에서는 document.write() HTML 출력 스트림에 직접 쓰는 데 사용할 수 있음

documemt.write()문서가 로드 된 후 사용하지 마요. 문서를 덮어 씀

HTML 내용 변경하기

HTML 요소의 내용을 수정하는 가장 쉬운 방법은 innerHTML 속성을 사용하는 것

document.getElementById(id).innerHTML = new HTML

속성 값 변경

document.getElementById(id).attribute = new value

요소의 src 특성 값을 변경함

document.getElementById("image").src = "landscape.jpg";
