자바 스크립트 출력

자바 스크립트 디스플레이 가능성

innerHTML을 사용하여 HTML 요소에 작성하기

document.write()를 사용하여 HTML 출력에 쓰기

window.alert()를 사용하여 경고 상자에 쓰기

console.log()를 사용하여 브라우저 콘솔에 기록하기

document.getElementById("demo").innerHTML = 5+6;

html요소의 innerHTML 속성을 변경하는 것은 데이터를 HTML로 표시하는 일반적인 방법

document.write(5+6);

테스트 목적으로 document.write()를 사용하는 것이 편리함.테스트 목적으로만 사용해야함!

HTML 문서가로드 된 후 document.write()를 사용하면 기존 HTML을 모두 삭제함

window.alert(5+6);

경고 상자를 사용하여 데이터를 표시 할 수 있습니다.

console.log(5+6)

디버깅을 위해 console.log()메소드를 사용하여 데이터를 표시 할 수 있음.
