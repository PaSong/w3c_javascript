자바스크립트 번호

자바스크립트에는 한 가지 유형의 번호 만 있음. 숫자는 십진수의 유무와 상관없이 쓸 수 있음. var x = 123e5; var y =123e-5;

숫자는 항상 64비트 부동 소수점임.

정수는 최대 15자리까지 정확

소수점은 최대 수는 17이지만 부동 소수점 산술은 항상 100% 정확하지 않음(문제를 해결하려면 곱셈 및 나눗셈을 수행하는 것이 좋음)

숫자 및 문자열 추가

숫자 + 숫자 = 숫자

문자 + 문자 = 문자

숫자 + 문자 = 문자

문자 + 숫자 = 문자

숫자 문자열

자바스크립트 문자열은 숫자로 된 내용을 가질 수 있음

자바스크립트에서 모든 숫자 연산에서 문자열을 숫자로 변환하려고 시도함

var z = x/y

var z = x*y

var z = x-y

var z = x+y (하지만 이것은 계산 안됨/문자열로 연결시킴)

NaN - 숫자가 아님

NaN은 숫자가 유효한 숫자가 아님을 나타내는 JavaScript 예약어

숫자가 아닌 문자열로 산술 연산을 수행하면 NaN(숫자가 아님)이 뜸.

무한대

Infinity는 가능한 가장 큰 숫자 이외의 숫자를 계산할 때 JavaScript가 반환하는 값
