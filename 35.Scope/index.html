Scope 범위

범위는 변수의 접근 가능성(가시성)을 결정

JavaScript 함수 범위

JavaScript에는 두 가지 유형의 범위가 있음

  • 로컬 범위
  • 전체 범위

JavaScript에는 기능 범위가 있음.각 기능은 새로운 범위를 만듬

범위는 이러한 변수의 접근 가능성(가시성)을 결정함.

함수 내에서 정의 된 변수는 함수 외부에서 액세스 (볼 수 없음)되지 않음

지역 JavaScript 변수

JavaScript 함수 내에서 선언 된 변수는 함수의 LOCAL이 됨.

지역 변수는 함수 번위를 가짐: 함수 내 에서만 액세스 할 수 있음

myFunction(); function myFunction(){ var carName = "Volvo"; document.getElementById("demo").innerHTML = typeof carName + "" + carName; } document.getElementById("demo2").innerHTML = typeof carName;

전역 JavaScript변수

함수 밖에서 선언 된 변수 GLOBAL이 됨

글로벌 변수가 글로벌 범위: 웹 페이지의 모든 스크립트와 기능에 액세스 할 수 있음.

var carName = "Volvo"; myFunction(); function myFunction() { document.getElementById("demo").innerHTML = "I can display" + carName; }

자바 스크립트 변수

JavaScript에서는 객체와 함수도 변수임

범위는 코드의 다른 부분에서 변수,객체 및 함수의 액세스 가능성을 결정함.

자동 글로벌

선언되지 않은 변수에 값을 할당하면 변수가 자동으로 GLOBAL변수가 됨.

myFunction(); //code here can use carName as a global variable document.getElementById("demo").innerHTML = "T can diplay" + carName; function myFunction(){ carName = "Volvo"; }

엄격모드

최신의 모든 브라우저는 "Stric MOde"에서 JavaScript를 실행할 수 있도록 지원함.

전역 변수는 "엄격 모드"에서 자동으로 생성되지 않음

HTML의 전역 변수

JavaScript를 사용하면 전역 범위가 완벽한 JavaScript 환경임.

HTML에서 전역 범위는 윈도우 객체임. 모든 전역 변수는 윈도우 객체에 속함.

var carName = "Volvo"; //code here can use window.carName; document.getElementById("demo").innerHTML = "I can display" + window.carName;

경고

의도하지 않은 한 전역 변수를 작성하지 마십시오.

전역 번수 (또는 함수)는 윈도우 변수(또는 함수)를 덮어 쓸 수 있음

창 개체를 포함한 모든 함수는 전역 번수 및 함수를 덮어 쓸 수 있음.

JavaScript 변수의 수명

JavaScript 변수의 수명은 선언 될 때 시작함.

함수가 완료되면 지역 변수가 삭제됨.

웹 브라우저에서 전역 변수는 브라우저 창 (또는 탭)을 닫을 때 삭제되지만 같은 창에 로드 된 새 페이지에서는 계속 사용할 수 있음.

함수 인수

함수 인수(매개 변수)는 함수 내에서 로컬 변수로 작동함
