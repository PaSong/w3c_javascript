Window Location

window.location 객체는 현재 페이지 주소(URL)를 얻기 위해 새로운 페이지로 브라우저를 리디렉션 할 수 있음

창 위치

window.location객체는 위도우 접두사 없이 작성 할 수 있음

window.location.href 현재 페이지의 href(URL)를 반환함

window.location.hostname 웹 호슽의 도메인 이름을 반환함

window.location.pathname 현재 페이지의 경로와 파일 이름을 반화함

window.location.protocal 사용된 웹 프로토콜 (http : 또는 https :)을 반환함

window.location.assign 새 문서를 로드함

창 위치 Href

window.location.href 속성은 현재 페이지의 URL을 반환함

document.getElementById("demo").innerHTML = "Page location is" + window.location.href;

창 위치 호스트 이름

window.location.hostname 속성은 (현재 페이지의) 인터넷 호스트의 이름을 반환함
