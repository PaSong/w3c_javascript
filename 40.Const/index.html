Const

const로 정의 된 변수는 재 할당 할 수 없다는 점을 제외하고 let변수와 같이 동작함

try{ const PI = 3.141592653589793; PI = 3.14; } catch(err){ document.getElementById("demo").innerHTML = err; }

블록범위

const를 사용하여 변수를 선언하는 것은 블록 범위와 관련하여 let과 비슷

var x =10; //Here x is 10 { const x =2; //Here x is 2 } //Here x is 10

선언되었을 때 할당 됨

JavaScript const 변수는 선언 될 때 값을 할당해야함

잘못된 예

const PI; PI = 3.14159265359;

옳은 예

const PI = 3.14159265359;

실제 상수가 아님

상수 값을 정의하지 않음. 값에 대한 상수 참조를 정의함

이 때문에, 우리는 상수 프리미티브 값을 변경할 수는 없지만 상수 객체의 프로퍼티를 변경할 수 있음

프리미티브 값

상수에 프리미티브 값을 할당하면 프리미티브 값을 변경할 수 없음

try{ const PI = 3.141592653589793; PI = 3.14; } catch(err){ document.getElementById("demo").innerHTML = err; } //TypeError:Assignment to constant variable.

상수 개체 변경 가능

상수 개체의 속성을 변경할 수 있음

//You can create a const object: cost car = {type:"Fiat", model:"500", color:"white"}; //You can change a property: car.color = "red"; //You can add a property: car.owner = "Johnson"; //Displlay the property: document.getElementById("demo").innerHTML = "Car owner is"+car.owner;

하지만 상수 객체를 재 할당 할 수는 없음

try{ const car = {type:"Fiat", model:"500", color:"white"}; car = {type:"Volvo", model:"EX60",color:"red"}; } cat (err){ document.getElementById("demo").innerHTML = err; }

상수 배열 변경 가능

상수 배열의 요소를 변경할 수 있음

//Create an Attay: const cars = ["Saab", "Volvo", "BMW"]; //Change an element: cars[0] = "Toyota"; //Add an element: cars.push("Audi"); //Display the Array: document.getElementById("demo").innerHTML = cars; // Toyota,Volvo,BMW,Audi

하지만 상수 배열을 재 할당 할 수는 없음

try{ const cars =["Saab","Volvo","BMW"]; cars = ["Toyota", "Volvo", "Audi"]; } catch(err){ document.getElementById("demo").innerHTML = err; } //error

재선언

JavaScript var 변수를 다시 선언하는 것은 프로그램의 어느 위치에서나 허용

기존 var 또는 let변수를 const, 동일한 범위 또는 동일한 블록에 다시 선언하거나 재 할당하는것은 허용되지 않음

var x =2; // Allowed const x =2; //Not Allowed { let x =2; //Allowed const x = 2; // Not allowed }

동일한 범위 또는 동일한 블록에서 기존의 const변수를 다시 선언하거나 재 할당하는것은 허용되지 않음

const x =2; //Allowed const x =3; //Not allowed x = 3; //Not allowed var x = 3; //Not allowed let x = 3; // Not allowed { const x = 2; //Allowed const x =3; //Not allowed x = 3; //Not allowed var x = 3; //Not allowed let x = 3; //Not Allowed }

const, 다른 범위 또는 다른 블록에서 변수를 다시 선언하는 것은 허용됨

const x =2; //Allowed { const x =3; //Allowed } { const x =4; //Allowed }

호이스팅

var로 정의된 변수는 맨 위에 올림

var변수를 선언하기 전에 사용할 수 있음

const로 정의된 변수는 맨 위로 올리지 않음

const가 선언되기 전에 변수를 사용할 수 없음
