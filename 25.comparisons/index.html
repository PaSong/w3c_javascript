조건부 (3진) 연산자

자바스크립트에는 조건에 따라 변수에 값을 할당하는 조건부 연산자가 포함되어 있음.

variablename = (condition) ? value1:value2

function myFunction(){ var age,voteable; age = document.getElementById("age").value; voteable = (age < 18) ? "Too young" : "Old enough"; document.getElementById("demo").innerHTML = voteable + "to vote." }

다른 유형 비교

서로 다른 유형의 데이터를 비교하면 예기치 않은 결과가 발생할 수 있음.

문자열과 숫자와 비교할때 자바스크립트는 비교를 수행 할 때 문자열로 변환.

빈 문자열은 0으로 변환.

숫자가 아닌 문자열은 항상 false인 NaN으로 변환 됨.

두 문자열을 비교할 때 "2"는 "12"보다 커야함.(알파벳 순으로) 1이 2보다 작기 때문

적절한 결과를 얻으려면 변수를 비교하기 전에 적절한 유형으로 변환해야함.

age = Number(age);

if (isNaN(age)){

voteable = "Input is not a number";

} else {

voteable = (age < 18) ? "Too young" : "Old enough";

}
