스위치 선언문

switch문은 다른 조건에 따라 다른 동장을 수행하는 데 사용됨.

swich(expression){ case x: //code block break; case y: //code block break; default: //code block }

  • 스위치 표현식은 한 번 평가됨.
  • 표현식의 값은 각 경우의 값과 비교됨
  • 일치하는 항목이 있으면 연관된 코드 블록이 실행

var day; switch(new Date().getDay()){ case 0: day = "Sunday"; break; case 1: day = "Monday"; break; case 2: day = "Monday"; break; case 3: day = "Monday"; break; case 4: day = "Monday"; break; case 5: day = "Monday"; break; case 6: day = "Monday"; break; } document.getElementById("demo").innerHTML = "Today is" + day;

break 키워드

break 키워드가 도달 하면 스위치 블록에서 빠져 나옴.

break는 스위치 블록에 있는 나머지 모든 코드의 실행을 "무시"하기 때문에 많은 실행 시간을 절약 할 수 있음.

var text; switch (new Date().getDay()){ case 6: text = "Today is Saturday"; break; case 0: text = "Today is Sunday"; break; default: text = "Looking forward to th Weekend"; } document.getElementById("demo").innerHTML = text;

공통 코드 블록

다른 스위치 케이스에서 동일한 코드를 사용하기 원할 때도 있음

switch(new Date().getDay()){ case 4: case 5: text = "Soon it is Weekend"; break; case 0: case 6: text = "It is Weekend"; break; default: text = "Looking forward to the Weekend"; }

세부 사항 전환

여러 사례가 사례값과 일치하면 첫 번쨰 사례가 선택됨

일치하는 사례가 없으면 프로그램은 기본 레이블을 계속 사용

기본 레이블이 없으면 프로그램은 전환 후 명령문으 계속 사용

엄격한 비교

스위치 케이스는 엄격한 비교 (===)를 사용함.

값은 일치해야하는 동일한 유형이야함.

엄격한 비교는 피연산자가 동일한 유형 인 경우에만 true가 될 수 있음.
