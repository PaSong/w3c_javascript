what is this

자바 스크립트 this 키워드는 이 속한 객체를 참조

this 값은 사용되는 위치에 따라 다른 값을 가짐.
  • 메소드에서 this는 소유자 객체를 참조 함.
  • 혼자,this는 전역 객체를 참조 함
  • 함수에서 this는 전역책체를 참조 함.
  • 함수에서 엄격한 모드에서는 this 안됨
  • 이벤트에서 this는 이벤트를 수신 한 요소를 나타냄
  • call() 및 apply()와 같은 메소드는 this를 모든 객체로 참조 할 수있음

this 하는 Method에서

객체 메소드에서 this는 메소드의 "소유자"를 참조함

이 페이지의 맨 위에 예에서, this받는 의미 인 개체

this 혼자

단독으로 사용하는 경우, 소유자는 전역 객체이기 때문에 this 글로벌 객체를 참조함.

브라우저 창에서 전역 개체는 [개체 창]임/

var x = this; document.getElementById("demo").innerHTML = x; //[objdect Window]

strict 모드 단독으로 사용되는 경우, this 또한 전역객체 [object Window]

this 기능(기본값)

자바 스크립트 함수에서 함수의 소유자임. 기본 바인딩 this

document.getElementById("demo").innerHTML = myFunction(); function myFunction(){ return this; } //[object Window]

this 기능(엄격)

자바 스크립트 strict 모드는 기본 바인딩을 허용하지 않음

함수에서 사용할 때, strict모드에서 this임. 정의되지 않음

"use strict"; document.getElementById("demo").innerHTML = myFunction(); function myFunction(){ return this; } // undefined

명시 적 함수 바인딩

그것들은 둘 다 인자로서 다른 객체를 가진 객체 메소드를 호출하는데 사용될 수 있음
