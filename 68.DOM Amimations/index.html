DOM Animation

자바스크립트를 사용하여 HTML 애니메이션을 만드는 법을 배움

요소 스타일 지정

컨테이너 요소는 style = "position : relative"으로 작성

애니메이션 요소는 style = "position: absolute"으로 작성

애니메이션 코드

자바스크립트 애니메이션은 요소 스타일의 점진적 변경을 프로그래밍하여 수행됨

변경 사항은 타이머에 의해 호출됨

타이머 간격이 작으면 애니메이션이 연속적으로 보임

var id = setInterval(frame, 5); function frame() { if( /* test for finished */) { clearInterval(id); } else { /* code to change the element style */ } }
