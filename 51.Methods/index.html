this 키워드

함수 정의에서 this 함수의 "소유자"를 참조함

위의 예에서 this는 IS인 목적은 "소유"한 FullName의 기능

즉 this.firstName은 this객체의 firstName 속성을 의미함

자바 스크립트 메소드

자바스크립트 메소드는 객체에서 수행 할 수 있는 액션임

JavaScript메소드는 함수 정의를 포함하느 속ㄴ성

메서드는 객체 속성으로 저장되는 함수임

객체 메소드 액세스

objectName.methodName()

일반적으로 person객체의 메서드로 fullName()을 속성으로 fullName을 설명함

fullName 속성은()로 호출 될 때(함수로) 실행됨

var person={ firstName: "John", lastName : "Doe", id : 5566, fullName : function(){ return this.firstName + " " + this.lastName; } }; document.getElementById("demo").innerHTML = person.fullName(); // John Doe

without()를 사용하여 fullName속성에 엑세스 하면 함수정의가 반환됨

var person ={ firstName : "John", lastName : "Doe", id : 5566, fullName : function(){ return this.firstName + " " + this.lastName; } }; document.getElementById("demo").innerHTML = person.fullName; // 함수정의가 반환됨
