문자열 메서드 및 속성

문자열 메서드는 문자열 작업을 도와줌

문자열 길이

var sln =txt.length;

문자열의 길이를 반환

문자열에서 문자열 찾기

var pos = str.indexOf('locate');

indexOf() 메소드(위치)의 인덱스를 반환.제 문자열에서 지정된 문자의 발생

lastIndexOf()에서도 ()메소드의 인덱스 반환 마지막 문자열에서 지정된 텍스트의 발생.

indexOf()및 lastIndexOf()는 텍스트가 없으면 -1을 반환

var pos = str.indexOf("locate",15)

문자열에서 문자열 검색

var pos = str.search("locate");

검색()메소드는 지정된 값에 대한 문자열을 검색하고 일치의 위치를 반환함

여기서 잠깐

search() 와 indexOf()는 동일하지 않음.

search() 메소드는 두 번째 시작 위치 인수를 취할 수 없음.

indexOf()메소드는 강력한 검색 값(정규표현식)을 사용할 수 없음

문자열 파트 추출하기

slice()

substring()

substr()

slice()메서드

var res = str.slice(7,13;)

문자열의 일부를 위치 7에서 위치 13으로 분할함

매개 변수가 음수이면 위치는 문자열 끝에서부터 계산

두 번째 매개 변수를 생략하면 메서드는 문자열의 나머지 부분을 잘라냄

인터넷익스플로러8이하에서는 음수위치가 작동되지 않음

substring()메서드

음수인덱스를 사용할 수 없음.

나머지는 slice()와 동일

sybstr()메서드

두번째 매개 변수는 추출 된 부분의 길이를 지정한다는 것

나머지는 slice()와 동일

두번째 매개 변수를 생략하면 substr()은 나머지 문자열을 잘라냄

음수면 끝에서 부터

문자열 내용 바꾸기

replace()메소드는 호출 된 문자열을 변경하지 않음. 새문자열을 반환

var n = str.replace("Microsoft","W3Schools");

첫번째 일치 항목 만 바꿈

대소문자을 구분

대소문자를 구분하지 않으려면 i플래그(정규식 없음) 사용하는 정규표현식을 사용

정규식은 따옴표 없이 작성

모든 일치 항목을 바꾸려면 /g 플래그(전역 일치)와 함께 정규식 사용

var n = str.replace(/Microsoft/g,"W3School");
