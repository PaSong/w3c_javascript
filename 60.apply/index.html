자바스크립트 기능 적용

방법 재사용

apply()메서드를 사용하면 다른 개체에 사용할 수 있는 메서드를 작성할 수 있음

자바스크립트 apply()메소드

apply()방버븐 call()방법과 유사함

var person = { fullName: function() { return this.firstName + " " + this.lastName; } } var person1 = { firstName:"John", lastName:"Doe", } var x = person.fullName.apply(person1); document.getElementById("demo").innerHTML = x;

call()과 apply()사이의 차이점

call()메소드는 인수를 별도로 취함

apply()메서드는 인수를 배열로 사용함

apply()메서드는 인수 목록 대신 배열을 사용하려는 경우 매우 편리함

인수를 사용한 apply()메소드

apply()메서드는 배열의 인수를 받아들임

var person ={ fullName: function(city, country) { return this.firstName + " " + this.lastName + "," + city + "," + country; } } var person1 ={ firstName:"John", lastName: "Doe", } var x = person.fullName.apply(person1, ["Oslo","Norway"]); document.getElementById("demo").innerHTML = x;

배열에 대한 최대 방법 시뮬레이션

Math.max()방법을 사용하여 숫자 목록에서 가장 큰 숫자를 찾을 수 있음

document.getElementById("demo").innerHTML = Math.max(1,2,3); // 3

자바스크립트 배열에는 max()메서드가 없으므로 Math.max()대신 메서드를 적용 할 수 있음

document.getElementById("demo").innerHTML = Math.max.apply(null, [1,2,3]); // 3 첫번쨰 인수 null은 즁요하지 않음 Math,' ', 0 이 들어갈 수 있음
