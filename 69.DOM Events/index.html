HTML DOM Events

이벤트에 반응하기

자바스크립트는 사용자가 HTML요소를 클릭 할 때처럼 이벤트가 발생할 때 실행될 수 있음

사용자가 요소를 클릭 할 때 코드를 실행하려면 자바스크립트 코드를 HTML 이벤트 속성에 추가 함.

HTML 이벤트의 예

  • 사용자가 마우스를 클릭하면
  • 웹페이지가 로드 되었을 때
  • 이미지가 로드되면
  • 마우스가 요소 위로 움직일 때
  • 입력 필드가 변경되면
  • HTML 약식이 제출되면
  • 사용자가 키를 칠 때

onload 및 onunload 이벤트

onload 및 onunload 이벤트는 사용자가 페이지에 들어가거나 페이지를 떠날 때 트리거 됨

onload 이벤트는 방문자의 브라우저 유형과 브라우저 버전을 확인하고 정보를 기반으로 적절한 버전의 웹 페이지를 로드하는데 사용할 수 있음

onload 및 onunload 이벤트는 쿠키를 처리하는 데 사용할 수 있음

function checkCookies() { var text = ""; if (navigator.cookieEnabled == true) { text = "Cookies are enabled."; } elese { text = "Cookies are not enabled."; } document.getElementById("demo").innerHTML = text; }
