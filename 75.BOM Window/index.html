The Browser Object Model

BOM(Browser Object Model)을 사용하면 JavaScript가 브라우저와 "대화"할 수 있음

브라우저 개체 모델(BOM)

공식적인 기준이 없음

최신 브라우저는 JavaScript 상호 작용에 대해 (거의) 동일한 메소드 및 속성을 구현했기 때문에 BOM의 메소드 및 속성으로 종종 언급됨

윈도우 객체

window 객체는 모든 브라우저에서 지원됨. 브라우저 창이 나타남

모든 전역 JavaScript 객체, 함수 및 변수는 자동으로 윈도우 객체의 멤버가 됨

전역 변수는 윈도우 객체의 속성임

전역 함수는 창 개체의 메서드임

심지어 (HTML DOM의)문서 객체는 윈도우 객체의 속성임

window.document.getElementById("header");

document.getElementById("header");

위에 두개는 같은 거임.

창 크기

두 개의 등록 정보를 사용하여 브라우저 창의 크기를 결정할 수 있음

두 속성 모두 크기를 픽셀 단위로 반환

window.innerHeight - 브라우저 창의 내부 높이 (픽셀 단위)

window.innerWidth - 브라우저 윈도우의 내부 너비 (픽셀 단위)

브라우저 창(브라우저 뷰포트)에는 도구 모음 및 스크롤 막대가 포함되지 않음

Internet Explorer 8,7,6,5:

document.documentElement.clientHeight

document.documentElement.clientWidth

document.body.clientHeight

document.body.clientWidth

다른 윈도우 메소드

window.open() - 새창을 염

window.close() - 현재 창 닫기

window.movwTo() - 현재 창 이동

window.resizeTo() - 현재 창 크기 조정
