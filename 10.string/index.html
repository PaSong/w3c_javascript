문자열

따옴표 안에 0개 이상의 문자로 작성

문자열 길이

문자열의 길이는 내장 된 속성 길이에서 찾을 수 있음

var txt ="abcdefg"

var sln = txt.length;

특수문자

백 슬래시(\) 이스케이프 문자는 특수 문자를 문자열 문자로 변환함.

\' = '

\" = "

\\ = \

긴 코드 라인 깨기

document.getElmentById("demo").innerHTML = "Hello" +

"Dolly!";

문자열을 분리하는 더 안전한 방법은 문자열 추가를 사용하는 것

백 슬래시를 사용하여 코드 줄을 분리 할 수 없음.

문자열은 객체가 될 수 있음

일반적으롤 자바스크립트 문자열은 리터럴에서 생성 된 프리미티브 값임.

var firstName = "John"

그러나 문자열은 키워드 new를 사용하여 객체로 정의 할 수도 있음

var firstName = new String("john");

문자열을 객체로 생성하지 마십시오. 실행 속도가 느려짐. new키워드는 코드를 복잡하게 함.이로인해 예상치 못한 결과가 발생할 수 있음
