JavaScript Statements

html에서는 자바 스크립트 프로그램이 웹 브라우저에 의해 실행됨.

명령문은 작성된 순서대로 하나씩 실행됨.

JavaScript 프로그램(및 JavaScript 문)은 종종 JavaScript 코드라고 합니다.

자바스크립트 문은 세미콜론으로 구분.

자바 스크립트 공백

자바 스크립트는 여러 공백을 무시함. 더 읽기 쉽게 공백을 추가할 수 있음.

JavaScript행 길이 및 줄 바꿈

가독성을 극대화하기 위해 프로그래머는 종종 80자 이상의 코드 행을 피하기 원함.

javaScript 코드 블록

자바스크립트 문은 중괄호{...}안에 코드 블록으로 그룹화 할 수 있음.

코드 블록의 목적은 함께 실행될 명령문을 정의하는 것

블록으로 그룹화 된 구문을 찾을 수 있는 곳은 JavaScript함수임.
