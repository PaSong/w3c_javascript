Boolean

true 또는 false의 두 값 중 하나를 나타냄

boolean값

매우 자주 프로그래밍에서 두 가지 값 중 하나만 가질 수 있는 데이터 유형이 필요함.

  • yes/no
  • no/off
  • TRUE/FALSE

Boolean()함수

함수를 사용하여 표현식(또는 변수)이 참인지 확인할 수 있음.

Boolean(10>9) //return true

또는 더 쉽게

(10>9) // also returns true

10>9 // also returns true

비교 및 조건

  • ==
  • >
  • <

"가치"가 있는 모든 것은 참임.

"가치"가 없는 모든 것은 거짓임.

  • 부울 값 0(영)은 false임.
  • -0
  • ""(빈문자열)
  • undefined
  • null
  • false
  • NaN

개체가 될 수 있는 불린

일반적으로 자바 스크립트 부울은 리터럴에서 생성 된 프리미티브 값임.

var x = false;

그러나 booleans는 키워드 new를 사용하여 객체로 정의할 수 있음.

var y = new (false);

var x = false; //typeof x returns boolean

var y = new Boolean(false); returns object

Boolean 객체를 만들지마오. 실행 속도가 느려짐. new 키워드는 코드를 복잡하게 함.

== 연산자를 사용하면 동등한 부울 값은 같음.

var x = false; // x is boolean

var y = new Boolean(false); //y is an object

document.getElementById("demo").innerHTML = (x == y); // true

=== 연산자를 사용할 때 ==연산자는 형식과 값에서 동등성을 기대하므로 동등한 블린은 동일하지 않음.

var x = false;

var y = new Boolean(false); // return false

(x == y)와 (x === y)의 차이점에 유의함. 두 개의 자바스크립트 객체를 비교하면 항상 false가 반환됨.
