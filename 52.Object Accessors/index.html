자바 스크립트 접근 자 (Geeters 및 Setters)

ECMAScript5는 게터 및 세터를 소개함

getter, sette를 사용하면 Object Accessors를 정의 할 수 있음

자바 스크립트 게터(Get keyword)

var person ={ firstName : "John", lastName : "Doe", language : "en", get lang(){ return this.language; } }; //Display data form the object using a getter: document.getElementById("demo").innerHTML = person.lang; //en

Setter(집합 키워드)

var person ={ firstName : "John", lastName : "Doe", language : "", set lang(lang){ this.language = lang; } }; //Set an object property using a setter: person.lang = "en" //Display data from the object: document.getElementById("demo").innerHTML = person.language; //en

데이터 품질

getter 및 setter를 사용할 때 더 나은 데이터 품질을 보장 할 수 있음.

lang 속성을 사용하면 language 속성의 값으 대문자로 반환함

게터와 세터를 사용하는 이유

단순한 구문을 제공함

속성과 메서드에 대해 동일한 구문을 사용할 수 있음

더 나은 데이터 품질을 확보 할 수 있음

비하인드 스토리를 만드는데 유용함

Object.defineProperty()

Getter 및 Setters를 추가 할 수 도 있음

var obj = {counter : 0}; Object.defineProperty(obj,"reset",{ get:function () {this.counter = 0;} }); Object.defineProperty(obj, "increment",{ get :function () {this.count++;} }); //Play with counter: obj.reset; obj.add = 5;
