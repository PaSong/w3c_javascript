random

Math.random()

Math.random()은 0(포함)과 1 (포함하지 않음) 사이의 난수를 반환

Math.random()은 항상 1보다 낮은 수를 반환함.

자바 스크립트 무작위 정수

Math.random()은 Math.floor()와 함께 사용되어 임의의 정수를 반환 할 수 있음.

Math.floor(Math.random()*10); //return a random integer from 0 to 9

적절한 임의 함수

모든 임의의 정수 용도로 사용할 적절한 임의 함수를 만드는 것이 좋음.

JavaScript 함수는 항상 min(포함됨)과 max(제외됨) 사이의 난수를 반환.

function getRndInteger(min, max){ return Math.floor(Math.random()*(max-min)) + min; }

JavaScript 함수는 항상 min과 max 사이의 난수를 반환함.(둘 다 포함)

function getRndInteger(min,max){ return Math.floor(Math.random()*(max - min +1)) + min; }
