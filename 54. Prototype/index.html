Prototype

프로토 타입 상속

모든 자바스크립트 객체는 프로토 타입에서 속성과 메서드를 상속받음

Date 객체는 Date.prototype에서 상속받음. Array객체는 Array.prototype에서 상속 받음

Person 객체는 Person.prototype에서 상속 받음

object.prototype은 프로토 타임 상속 체인의 맨 위에 있음

Date 객체, Array 객체 및 Person 객체는 Object.prototype에서 상속받음

객체에 속성 및 메서드 추가

Using 프로토 타입 속성

자바스크립트 프로토 타입속성을 사용하면 객체 생성자에 새 속성을 추가 할 수 있음

function Person(first, last, age, eye) { this.firstName = first; this.lastName = last; this.age = age; this.eyeColor = eye; } Person.prototyoe.nationality = "English";
