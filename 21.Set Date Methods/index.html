설정 날짜 메소드

날짜 설정 메서드를 사용하면 날짜 객체의 날짜 값(년,월,일,시간,분,초,밀리 초)을 설정할 수 있음.

  • setDate()
  • setFullYear()
  • setHours()
  • setMilliseconds()
  • setMinutes()
  • setMonth()
  • setSeconds()
  • setTime()

setFullYear()메서드

Date객체의 연도를 설정함

var d = new Date();

d.setFfullYear(2020);

document.getElementById("demo").innerHTML = d;

setFullYear()메서드는 선택적으로 월과 일을 설정할 수 있음.

var d = new Date();

d.setFullYear(2020,11,3);

document.getElementById("demo").innerHTML = d;

setMonth()메서드

날짜 개체(0-11)의 달을 설정(월을 변경할 수 있음)

var d = new Date();

d.setMonth(11);

document.getElementById("demo").innerHTML = d;

setDate()메서드

setDate()메서드는 날짜 개체(1-31)의 일을 설정함.

var d = new Date();

d.setDate(20);

document.getElementById("demo").innerHTML = d;

setDate()메서드를 사용하여 날짜에 날짜를 추가 할 수 있음(더하기)

var d = new Date();

d.setDate(d.getDate() + 50);

document.getElementById("demo").innerHTML = d;

날짜를 추가하거나 월 또는 연도를 이동하면 변경 사항은 Date 객체에 의해 자동을 처리됨.

setHours()메서드

날짜 오브젝트(0-23)의 시간을 설정.

ver d = new Date();

d.sethours(22);

setMinutes()메서드

날짜 오브젝트(0-59)의 분을 설정

var d = new Date();

d.setMinutes(30);

document.getElementById("demo").innerHTML = d;

setSeconds()메서드

날짜 오브젝트 (0-59)의 초 정보를 설정

var d = new Date();

d.setSeconds(30);

document.getElementById("demo").innerHTML = d;
