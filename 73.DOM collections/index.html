DOM Collections

HTMLCollection 객체

getElementsByTagName()메서드는 반환 HTMLCollection객체를.

HTMLCollection객체는 HTML 요소의 배열과 같은 리스트(콜렉션)이다

var myCollection = document.getElementsByTagName("p"); document.getElementById("demo").innerHTML = "The innerHTML of the second paragraph is: " + myCollection[1].innerHTML;

HTML HTMLCollection길이

length속성은 다은 요소의 수를 정의함

var myCollection = document.getElementsByTagName("p"); document.getElementById("demo").innerHTML = "This document contains" + myCollection.length + "paragraphs.";

function myFunction() { var myCollection = document.getElementsByTagName("p"); var i; for (i = 0; i < myCollection.length; i++){ myCollection[i].style.color = "red"; } }

HTMLCollection은 배열이 아님. 배열처럼 보이지만 배열이 아님. 목록을 반목하면서 번호가 있는 요소를 참조 할 수 있음. 배열 메서드를 사용할 수 없음.
