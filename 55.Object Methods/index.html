Object Methods

속성 값 변경

object.defineProperty(object, property, {value : value})

var person ={ firstName : "John"; lastName : "Doe"; language : "EN" } Object.defineProperty(person, "language", {value:"NO"}) document.getElementById("demo").innerHTML = person.language; //NO

메타 데이터 변경

ES5를 사용하면 다음 속성 데이터를 변경할 수 있음

ES5를 사용하면 getter 및 setter를 변경할 수 있음

언어를 읽기 전용으로 만듬

Object.defineProperty(person, "language", {writable:false});

언어를 열거 할 수 없도록 만듬

Object.defineProperty(person, "language", {enumerable:false});

모든 속성 나열

var person ={ firstName : "John", lastName : "Doe", language : "EN" } Object.defineProperty(person, "language", {enumerable:false}); document.getElementById("demo").innerHTML = Object.getOwnPropertyName(person); //firstName,LastName,language
