Navigator

window.navigator 객체는 방문자의 브라우저에 대한 정보가 포함되어 있음

창 네비게이터

window.navigator 객체는 윈도우 접두자없이 작성 할 수 있음.

  • navigator.appName
  • navigator.appCodeName
  • navigator.platform

브라우저 쿠키

cookieEnabled쿠키가 활성화되면 이 속성은 true를 반환하고 그렇지 않으면 false를 반환함

document.getElementById("demo").innerHTML = "navigator.cookieEnabled is" + navigator.cokieEnabled;

브라우저 응용 프로그램 이름

appName속성은 브라우저의 응용 프로그램 이름을 반환함

document.getElementById("demo").innerHTML = "navigatro.appName is " + navigator.appName;

"네스케이프"는 IE11, 크롬, 파이어 폭스, 사파리의 애플리케이션 이름

브라우저 응용 프로그램 코드 이름

appCodeName 속성은 브라우저의 응용 프로그램 코드 이름을 반환

브라우저 엔진

product속성은 브라우저 엔젠의 제품 이름을 반환함

브라우저 버전

appVersion 속서은 브라우저에 대한 버전 정보를 반환함

브라우저 에이전트

userAgent속성은 브라우저에서 브라우저로 보낸 사용자 에이전트 헤더를 서버로 반환함

경고

탐색기 객체의 정보는 종종 오도 될 수 있으므로 다음과 같은 이유로 브라우저 버전을 감지하는데 사용하면 안됨

  • 여러 브라우저에서 동일한 이름을 사용할 수 있음
  • 탐색기 데이터는 브라우저 소유자가 변경할 수 있음
  • 일부 브라우저는 사이트 테스트를 우회하는 것으로 잘못 판단함
  • 브라우저는 브라우저보다 나중에 릴리스 된 새 운영 체제를 보고 할 수 없음

브라우저 플랫폼

platform 속성은 브라우저 플랫폼 (운영 체제)을 반환함
