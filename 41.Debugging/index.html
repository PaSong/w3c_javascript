디버깅

새로운 컴퓨터 코드를 작성할 때마다 오류가 발생할 수 있음

코드 디버깅

프로그래밍 코드에는 구문 오류 또는 논리 오루가 포함될 수 있음

프로그래밍 코드에 오류가 있는 경우 아무 일도 일어나지 않는 경우가 많음

오류 메세지가 없으므로 오류를 검색할 위치를 알 수 없음

프로그래밍 코드에서 오류를 찾고 수정하는 것을 코드 디버깅이라고 함

자바 스크립트 디버거

디버깅은 쉽지 않음. 최신브라우저에는 JavaScript 디버거가 내장되어 있음

내장 된 디버거는 켜고 끌 수 있어 사용자에게 오류를 보고

디버거를 사용하면 중단 점(코드 실행을 중지 할 수 있는 위치)을 설정하고

코드가 실행되는 동안 변수를 검사 할 수도 있음

console.log()메소드

중단 점 설정

디버거 창에서 JavasCript코드에 중단 점을 설정할 수 있음

각 중단점에서 JavaScript는 실행을 중지하고 JavaScript값을 검사 할 수 있음

값을 검사 한 후 코드 실행을 재개 할 수 있음(일반적으로 재생 버튼 사용)

디버거 키워드

디버거 키워드는 자바 스크립트의 실행 및 호출(가능한 경우)디버깅 기능을 중지함

이것은 디버거에서 중단 점을 설정하는 것과 같은 기능을 함

사용할 수 있는 디버깅이 없으면 디버거 문은 아무효과가 없음

var x = 15 * 5; debugger; document.getElementById("demo").innerHTML = x;
