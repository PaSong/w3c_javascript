HTML DOM메소드

HTML DOM메소드는 HTML 요소에서 수행 할 수 있는 작업임

HTML DOM 속성은 사용자가 설정하거나 변경할 수 있는 HTML 요소의 값임

DOM 프로그래밍 인터페이스

HTML DOM은 javaScript(및 다른 프로그래밍 언어)로 엑세스 할 수 있음

DOM에서 모든 HTML 요소는 객체로 정의됨

프로그래밍 인터페이스는 각 객체의 속성 및 메서드임

속성은 당신이 (HTML 요소의 내용을 변경하는 등) 가져 오거나 설정할 수 있는 값임

방법은 (추가하거나 HTML요소를 삭제 같은)당신이 할 수 있는 작업임

document.getElementById("demo").innerHTML = "Hello World!";

getElementById에서 메서드는 반면 innerHTML에 속성임

getElementById메서드

HTML요소에 액세스하는 가장 일반적인 방법 id은 the요소를 사용하는 것임

innerHTML속성

요소의 내용을 가져 오는 가장 쉬운 방법은 innerHTML속성을 사용하는 것임

innerHTML속성을 사용하면 을 포함한 모든 HTML 요소를 가져 오거나 변경할 수 있음
