HTML DOM

HTML DOM을 사용하면 자바스크립트가 HTML문서의 모든 요소에 엑세스하여 이를 변경 할 수 있음

HTML DOM(Document Object Model)

웹 페이지가 로드 될 때

객체 모델을 사용하면 JavaScript는 동적 HTML을 만드는데 필요한 모든 기능을 제공함

  • 자바스크립트는 페이지의 모든 HTML 요소를 변경할 수 있음
  • 자바스크립트는 페이지의 모든 HTML 속성을 변경할 수 있음
  • 자바스크립트는 페이지의 모든 css스타일을 변경할 수 있음
  • 자바스크립트는 기존 HTML 요소 및 속성을 제거 할 수 있음
  • 자바스크립트는 새로운 HTML 요소와 속성을 추가 할 수 있음
  • 자바스크립트는 페이지의 모든 기존 HTML 이벤트에 반응 할 수 있음
  • 자바스크립트는 페이지에서 새로운 HTML 이벤트를 생성 할 수 있음

DOM이란 무엇?

DOM W3C(World Wide Wdb Consortium) 표준임

DOM은 문서에 액세스하기 위한 표준을 정의함

W3C DOM(Document Object Model)은 프로그램 및 스크립트가 문서의 내용, 구조 및 스타일을 동적으로 액세스하고 업데이트 할 수 있게 해주는 플랫폼 및 언어 중립적 인터페이스임.

  • Core DOM - 모든 문서 유형에 대한 표준 모델
  • XML DOM - XML 문서의 표준 모델
  • HTML DOM - HTML 문서의 표준 모델

HTML DOM이란 무엇?

HTML DOM은 HTML에 대한 표준 객체 모델 및 프로그래밍 인터페이스임

  • 객체로서의 HTML요소
  • 모든 HTML 요소의 속성
  • 모든 HTML 요소에 액세스 하는 메소드
  • 이벤트의 모든 HTML요소

즉,HTML DOM은 HTML요소를 가져오 거나,변경하거나,추가하거나,삭제하는 방법의 표준
