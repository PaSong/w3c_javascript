break문

break문은 루프를 중단하고 루프(있는 경우)후 코드를 계속 실행

var text = ""; var i; for (i = 0; i < 10; i++){ if(i === 3){break;} text += "The number is" + i + "
"; } document.getElementById("demo").innerHTML = text;

continue문

지정된 조건이 발생하고 루프의 다음 반복을 계속하는 경우,(루프)에 하나의 반복 휴식을

var text = ""; var i; for (i = 0;i < 10; i++){ if ( i === 3){ continue; } text += "The number is" + i + "
" } document.getElementById("demo").innerHTML = text;

i가 3일 경우 3을 건너뛰고 계속 진행한다.

자바스크립트 라벨

JavaScript 문에 레이블을 지정하려면 문 앞에 레이블 이름과 콜론이 있어야함.

label : statements

break와 continue문은 코드 블록에서 "뛰어 넘을"수 있는 유일한 JavaScript문임.

break labelname;

continue labelname;

continuew문(레이블 참조 유무와 관계없이)은 루프반복을 건너 뛸 때만 사용할 수 있음.

레이블 참조가 없는 break문은 루프 또는 스위치 밖으로 점프하는 경우에만 사용할 수 있음

레이블 참조를 사용하면 break문을 사용하여 코드 블록에서 빠져 나올 수 있음.

var cars = ['bmw','volvo','saab','ford']; var text = ""; list: { text += cars[0] + "
" text += cars[1] + "
" break list; text += cars[2] + "
" text += cars[3] + "
" } document.getElementById("demo").innerHTML = text;
