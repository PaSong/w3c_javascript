Parameters 매개 변수

JavaScript function은 매개 변수 값(인수)을 검사하지 않음

함수 매개 변수 및 인수

function functionName(parameter1, parameter2, parameter3) { //code to be executed }

함수 매개변수는 함수 정의에 나열된 이름임

함수 인수는 함수에 전달된 실제 값임

매개 변수 규칙

함수 정의는 매개 변수에 대한 데이터 유형을 지정하지 않음

함수는 전달 된 인수에 대해 유형 검사를 수행하지 않음

함수는 받은 인수의 수를 확인하지 않음

매개 변수 기본값

누락 된 인수(선언 되지 않음)가 있는 함수가 호출 된 경우 누락 된 값은 undefined

때로는 이것이 허용 가능하지만 떄로는 매개 변수에 기본값을 지정하는 것이 좋음

function myFunction(x, y) { if (y === undefined) { y = 0; } return x * y; } document.getElementById("demo").innerHTML = myFunction(4); // 0

ECMAScript 2015는 함수 호출에서 기본 매개 변수를 허용함

function (a=1, b=1) { // function code }

Arguments 객체

자바스크립트 함수에는 arguments 객체라는 기본 제공 객체가 있음

인수 객체는 함수가 호출 될 때(호출 된) 사용 된 인수 배열을 포함함

이 방법을 사용하면 함수를 사용하여 숫자 목록에서 가장 높은 값을 찾을 수 있음

function findMax(){ var i; var max = -Infinity; for(i = 0; i < arguments.length; i++) { if (arguments[i] > max) { max = arguments[i]; } } return max; } document.getElementById("demo").innerHTML = findMax(4,5,6); //6

또는 모든 입력 값을 합산하는 함수를 만듬

function sumAll() { var i; var sum = 0; for(i = 0; i < arguments.length; i++) { sum += arguments[i]; } return sum; } document.getElementById("demo").innerHTML = sumAll(1, 123, 500, 115, 44, 88); // 871

너무 많은 인수(선언 된 것 이상)로 함수를 호출 하면 argument객체를 사용하여 이러한 인수에 연결할 수 있음

인수는 값으로 전달됨

함수 호출에서 매개 변수는 함수의 인수임

자바스클비트 인수는 값에 의해 전달됨. 함수는 인수의 위치가 아니라 값을 알 수 있음

함수가 인수의 값을 변경하면 매개 변수의 원래 값은 변경되지 않음

인수 변경은 함수 외부에 표시(반영)되지 않음

객체는 참조로 전달됨

자바스크립트에서 객체 참조는 값임

이 때문에 객체는 참조로 전달되는 것처럼 동작함

함수가 객체 속성을 변경하면 원래 값이 변경됨

객체 속성에 대한 변경 사항은 함수 외부에 표시 (반영)됨
