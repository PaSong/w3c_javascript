배열 메소드

배열을 문자열로 반환

toString()은 배열을 쉼표로 구분 된 배열 값의 문자열로 변환

join()메서드는 문자열의 모든 배열 요소를 결합함.

toString()과 똑같이 동작하지만, 구분자를 지정할 수도 있음.

var fruits = ["banana","orange","apple","mango"]; document.getElementById("demo").innerHTML = fruits.join("*"); // banan * orange * apple * mango

pop 과 push

배열을 사용하여 작업하면 요소를 쉽게 제거하고 새요소를 추가 할 수 있음.

pop()

pop()메소드는 배열의 마지막 한 요소를 제거함.

pop()메서드는 값을 반환함

push()메소드는 array에 새로운 요소를 추가함.

push() 메서드는 새로운 배열 길이를 반환

요소이동

shift은 popping과 동일하며 마지막 요소 대신 첫 번쨰 요소에서 작업함

shift()방법은 첫번째 요소를 제거

shift()메서드는 "이동 된" 문자열을 반환

unshift()는 배열 처음에 새로운 요소를 추가

unshift()메서드는 새로운 배열 길이를 반환

요소 변경하기

배열 요소는 해당 인덱스 번호를 사용하여 액세스됨.

var fruits = ["banana","orange","apple","mango"]; fruits[0] = "kiwi";

length속성은 배열에 새 요소를 추가하는 쉬운 방법을 제공

var fruits = ["banana","orange","apple","mango"]; fruits[fruits.length] = "kiwi";

요소 삭제하기

자바스크립트 배열은 객체이기 때문에 자바스크립트 연산자 delete를 사용하여 요소를 삭제할 수 있음.

var fruits =['banana','orange','apple','mango']; delete fruits[0];

delete를 사용하면 배열에 정의되지 않은 구멍이 생길 수 있음. 대신 pop()또는 shift()를 사용해.

배열 연결하기

splice()메소드는 array에 새로운 항목을 추가 할 수 있음.

var fruits =['banana','orange','apple','mango']; fruits.splice(2,0,"lemon","kiwi"); - (2) 첫번째 요소가 추가될 위치 인덱스 - (0) 제거 되어야 하는 요소 수 (첫번째부터) - 나머지 : 새로 추가될 요소

splice()를 사용하여 요소 제거

splice()를 사용하여 배열에 "구멍"을 남기지 않고 요소를 제거 할 수 있음

fruits.splice(0,1); -(0): 새 요소가 추가되야하는 위치 -(1): 제거 되어야 하는 요소 수를 정의

배열 병합(연결)

concat()메소드는 기존의 배열들을 병합하여 새로운 배열을 작성

var myChildren = myGirls.concat(myBoys);

concat()메서드는 기존 배열을 변경하지 않음. 항상 새로운 배열을 반환

concat()메소드는 여러 개의 배열 인수를 취할 수 있음.

var myChildren = arr1.concat(arr2,arr3);

concat()메서드는 값을 인수로 취할 수도 있음.

var myChildren = arr1.concat(["Emil","Tobias","Linus"]);

배열 자르기

var fruits =['banana','orange','apple','mango']; var citrus = fruits.slice(1); // orange apple mango

slice()메서드는 새로운 배열을 만듬. 소스 배열에서 요소를 제거하지 않음.

slice()메서드는 slice(1,3)와 같은 두 개의 인수를 사용할 수 있음

시작 인수에서 끝 인수까지 요소를 선택하지만 마지막 요소는 나오지 않음.

fruits.slice(2)

인덱스 2번째 부터 끝까지 나옴

자동 toString()

프리미티브 값이 예상되면 JavaScript는 자동으로 배열을 쉼표로 구분 된 문자열로 변환함.

모든 JavaScript 객체에서는 toString()메서드가 있음.

배열에서 최대 값과 최소값 찾기

자바스크립트 배열에서 가장 높은 값 또는 가장 낮은 값을 찾는 내장 함수는 없음.
