조건문

if(condition){ // block of code to be executed if the condition is true}

if는 소문자로 적어야함. 대문자로 적으면 오류를 일으킴

else문

조건이 거짓일 경우 코드 블록이 실행되도록 지정하는 문.

if (condition){ }else{}

else if문

첫번째 조건이 false인 경우 새 조건을 지정할 수 있음.

if(condition1){ }else if (condition2){ }else{}
