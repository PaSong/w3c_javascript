HTML DOM 탐색

HTML DOM을 사용하면 노드 관계를 사용하여 노드 트리를 탐색 할 수 있음

DOM 노드

HTML 문서의 모든 것이 노드임

  • 전체 문서가 문서 노드임
  • 모든 HTML 요소는 요소 노드임
  • HTML 요소 내부의 텍스트는 텍스트 노드임
  • 모든 HTML 속성은 속성 노드임(사용되지 않음)
  • 모든 주석은 주석 노드임

HTML DOM을 사용하면 노드 트리의 모든 노드에 JavaScript로 액세스 할 수 있음

새 노드를 만들 수 있으며 모든 노드를 수정하거나 삭제할 수 있음

노드 관계

노드 트리의 노드는 서로 계층 관계를 갖음

상위, 하위 및 형제라는 용어는 관계를 설명하는데 사용됨

  • 노드 트리에서 최상위 노드는 루트(또는 루트 노드)
  • 모든 노드에는 정확히 하나의 부모가 있지만 루트(부모가 없음)
  • 노드는 많은 수의 자식을 가질 수 있음
  • 형제(형제 또는 자매)는 같은 부모를 가진 노드임

노드 간 이동

노드 속성을 사용하여 JavaScript가 있는 노드 사이를 탐색 할 수 있음

  • parentNode
  • childNodes[nodenumber]
  • firstChild
  • lastchild
  • nextSibling
  • previousSibling

하위 노드 및 노드 값

DOM처리의 일반적인 오류는 요소 노드가 텍스트를 포함 할 것으로 기대하는 것임

innerHTML 속성에 액세스하는 nodeValue것은 첫 번째 자식에 액세스하는 것과 동일함

var myTitle = document.getElementById("demo").firstChild.nodeValue;

var myTitle = document.getElementById("demo").childNodes[0].nodeValue;

DOM 루트 노드

전체 문서에 대한 액세스를 허용하는 두 가지 특수 속성이 있음

  • document.body - 문서 본문
  • document.documentElement -전체 문서

nodeName 속성

nodeName등록 정보는 노드의 이름을 지정함

  • nodename은 읽기 전용임
  • 요소 노드의 nodeName은 태그 이름과 같음
  • 속성 노드의 nodeName은 속성 이름임
  • 텍스트 노드의 nodeName은 항상 #text임
  • 문서 노드의 nodeName은 항상 #document임
nodeName 항상 HTML 요소의 대문자 태그 이름이 포함됨

nodeValue 속성

nodeValue속성은 노드의 값을 지정함

  • 요소 노드에 대한 nodeValue는 null
  • 텍스트 노드의 nodeValue는 텍스트 자체임
  • 속성 노드의 nodeValue는 속성 값임

nodeType속성

nodeType속성은 읽기 전용임.노드의 유형을 리턴함
