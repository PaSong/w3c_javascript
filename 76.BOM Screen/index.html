자바 스크립트 창 화면

window.screen 객체는 사용자 화면에 대한 정보를 포함함

윈도우 화면

window.screen 객체는 윈도우 접두사없이 작성 할 수 있음

  • screen.width
  • secreen.height
  • screen.availHeight
  • screen.avaiWidth
  • screen.colorDepth
  • screen.pixelDepth

창 화면 너비

screen.width속성은 방문자의 화면 너비를 픽셀 단위로 반환함

document.getElement.ById("demo").innerHTML = "Screen width is" + screen.width;

창 화면 높이

screen.height속성은 방문자의 화면 높이를 픽셀 단위로 반환함

document.getElementById("demo").innerHTML = "Screen Height " + screen.height;

창 화면 사용 가능한 너비

screen.availWidth속성은 Windows 작업 표시 줄과 같은 인터페이스 기능을 제외한 방문자 화면의 너비를 픽셀 단위로 반환함

document.getElementById("demo").innerHTML = "Available Screen Width" + screen.availWidth;

창 화면 사용 가능한 높이

screen.availHeight속성은 방문자의 화면 높이(픽셀 단위)에서 Windows 작업 표시 줄과 같은 인터페이스 기능을 뺀 값을 반환함

창 화면 색 농도

screen.colorDepth속성은 하나의 색상을 표시하는데 사용되는 비트 수를 반환함.

현대의 모든 컴퓨터는 컬러 해상도를 위해 24비트 또는 32비트 하드웨어를 사용함

24비트 = 16,777,216 다른 "트루 컬러"

32비트 = 4,294,967,296 다른 "딥 컬러"

HTML에 사용된 #rrggbb(rgb) 값은 "True Colors"(16,777,216가지 색상)

창 화면 픽셀 깊이

screen.pixelDepth 속성은 화면의 픽셀 깊이를 반환함

최신 컴퓨터의 경우 색상 수와 픽셀 수 는 동일함
