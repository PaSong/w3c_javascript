Popup Boxes

자바 스크립트에는 경고 상자, 확인 상자 및 프롬프트 상자라는 세 종류의 팝업 상자가 있음

알림 상자

사용자에게 정보가 전달되도록하려면 경고 상자가 자주 사용됨

경고 상자가 나타나면 사용자가 계속 진행하기 위해 '확인'을 클릭해야함

window.alert("sometext");

window.alert()메서드는 윈도우 프리픽스 없이 쓸 수 있음

상자 확인

window.confirm("sometext");

윈도우 프리픽스 없이 쓸 수 있음

function myFunction(){ var txt; if (confirm("Press a button!")) { text = "You pressed OK!"; }else { txt ="You pressed Cancel!"; } document.getElementById("demo").innerHTML = txt; }

프롬프트 상자

사용자가 페이지에 들어가기 전에 값을 입력하게 하려면 프로프트 상자가 자주 사용됨

줄 바꿈

팝업 상자 안에 줄 바꿈을 표시하려면 백 슬래시 뒤에 문자 n을 사용해라

\n
